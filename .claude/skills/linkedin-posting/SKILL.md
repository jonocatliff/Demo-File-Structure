# LinkedIn Posting Skill

## What This Skill Does
Creates a high-performing LinkedIn post in Jono's voice about AI, automation, and Claude Code. Every post must pass a 9/10 quality gate before it's considered done.

## Inputs Needed Upfront
Ask Jono at the start:
1. **Topic/angle** — What's the post about? (e.g., a tip, a story, a hot take, a lesson learned)
2. **Key point** — What's the one thing the reader should walk away with?
3. **CTA goal** — What do you want people to do? (comment, connect, repost, DM, link click)

## How to Write the Post

### Step 1: Load Context
Read these files before writing anything:
- `references/winning-linkedin-posts.md` — Jono's proven post style and structure

### Step 2: Write the Post
Follow these rules based on what works in Jono's winning posts:

**Hook (first 2 lines — this is everything):**
- Must create curiosity or tension — the reader needs a reason to click "see more"
- Use bold unicode text for the opening line when it's a strong statement
- Specific numbers beat vague claims ("500+ hours" not "a long time")
- Pattern interrupt — say something unexpected or counterintuitive

**Body structure:**
- Short paragraphs (1-3 lines max)
- Use numbered lists for step-by-step content
- Use → arrows for key takeaways or bullet points
- Mix personal story with practical value — never just one
- Conversational tone — write like you're talking to one person
- Include a "turning point" moment (what changed, what you realized)

**CTA (last few lines):**
- Make it dead simple — one clear action
- Use the "want it?" pattern: ask a question, then tell them exactly what to do
- Comment-trigger CTAs work well (e.g., "Comment 'TIPS' and I'll DM you")
- Repost incentive as a bonus (e.g., "Repost this and I'll throw in X too")

**Voice rules (from Jono's style):**
- First person, always
- Self-aware and honest — admit mistakes, share the real journey
- No corporate speak, no buzzwords
- Emoji usage: minimal, purposeful (👉 for emphasis, not decoration)
- Use "Here's what..." and "The biggest mistake?" patterns
- Contractions always (don't, isn't, won't — never "do not")

### Step 3: Score the Post
Rate the post honestly on these criteria (must average 9/10+):
- **Hook strength** — Would YOU click "see more"? Is it specific, not generic?
- **Body structure** — Does it follow the winning formula? Easy to scan?
- **Originality** — Would someone screenshot this? Is there a fresh angle?
- **CTA clarity** — Is there exactly one clear action? Is it frictionless?
- **Voice match** — Does it sound like Jono, not a LinkedIn guru?

If it's below 9/10, say what's dragging the score down and fix it before presenting.

### Step 4: Present the Post
Show the final post in a copy-paste ready format (no code blocks, no markdown that won't render on LinkedIn). Include:
- The post text exactly as it should appear on LinkedIn
- The score breakdown
- One suggestion for an image/visual if relevant

## What NOT to Do
- Don't use hashtags (Jono doesn't use them)
- Don't write generic motivational content
- Don't open with "I'm excited to share..." or "Thrilled to announce..."
- Don't use more than 2 emojis per post
- Don't make the post longer than ~1500 characters unless the content demands it
- Don't add line breaks between every single line — group related thoughts
